### Installation

Hotel App requires [Node.js](https://nodejs.org/) and a virtual mobile device to run.

Install the dependencies.

```sh
cd HotelApp
npm i
```

For android app

```sh
npm run android
```

For iOS app

```sh
npm run ios
```
